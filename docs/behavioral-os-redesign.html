The non-negotiables

Design doctrine

DOC-01

The brief lists 23 persuasion levers. In a money app, half turn predatory the moment they're aimed at trade frequency. This doctrine re-aims every one at informedness — the only engagement that compounds trust instead of spending it.

P1

Reward the read, not the trade

Streaks, XP, and variable rewards attach to reviewing intelligence and logging decisions — never to placing orders. The daily win is “I understood the market,” gated behind zero transactions.

P2

Every claim shows its work

Authority is earned by transparency. Confidence numbers link to the TWRD source posterior and the calibration curve behind them. No black-box “87% BUY.”

P3

Calibration is the scoreboard

The core progression metric is Brier / reliability — “were your confident calls actually right?” Overconfidence costs points, aligning the dopamine loop with real skill.

P4

Loss aversion serves protection

Loss framing is allowed only pointed at risk avoided (Fortress, ODG scar blocks), never “you’ll miss out — buy now.” Scarcity is reserved for genuinely time-boxed events, always dated.

P5

The user can always turn it off

Every gamified layer has a “just the terminal” switch. Habit through value, retained by trust; a prosumer who feels manipulated churns to a spreadsheet.

The behavioral operating system

SYS·00

Eight persistent systems that live above the screens. Each maps to a module already in the codebase — this is instrumentation and surfacing, not new science. These are what the user carries between screens and between days.

DQ

Decision Quality Score

maps → useOutcomeGradient · ODG G_new

Every logged decision scored on process, not outcome: did you respect the drawdown gate, the scar memory, the regime? A good call that lost still scores well.

Trigger log decision → Action ODG grades → Reward DQ delta + reason → Investment streak++
CB

Calibration Belt

maps → reliabilityCurve · Brier (new)

A rank (Novice → Oracle) earned only by being right when you said you were sure. Powered by the calibration layer shipped in the quant upgrade.

Trigger prediction resolves → Action Brier update → Reward belt progress → Investment more predictions
SM

Scar Journal

maps → ScarMemory · odg-validator

Your ledger of failure patterns avoided and repeated. The Zeigarnik hook: an open scar stays pinned until you review the pattern that caused it.

Trigger scar match → Action read the past loss → Reward “pattern broken” → Investment endowment in history
PH

Portfolio Health

maps → quant-engine · Fortress · VaR

A single 0–100 vital sign (diversification, tail risk, regime-fit, crowding). Progress-bias done honestly: users return to keep the number green, a defensive habit.

Trigger health drifts → Action diagnose → Reward restored vital → Investment tuned portfolio
DB

Daily Briefing

maps → useQuantSnapshot · regime · CLANK

The anchor ritual. A one-screen, 90-second synthesis personalized to your holdings. Variable content = variable reward; same slot every morning = habit.

Trigger 06:45 push → Action read → Reward “ahead of the open” → Investment streak + saved cards
CF

Confidence Progression

maps → TWRD · veracity gate

Sources you trust build a track record you can see. TWRD’s Beta posteriors become “this analyst was right 63% on your names,” turning credibility into collectible reputation.

Trigger source resolves → Action posterior update → Reward trust delta → Investment curated sources
OD

Opportunity Discovery

maps → StatArb · CLANK · scanner

Milestones for finding, not chasing. “Spotted a CLANK cascade before it fired.” Curiosity-gap teasers reveal the reasoning, not just a ticker.

Trigger scan surfaces edge → Action inspect the why → Reward discovery badge → Investment saved thesis
FW

Future-Wealth Simulator

maps → FGM · Monte Carlo

Counters hyperbolic discounting: renders the probabilistic cone of where disciplined process leads. Median + P10/P90, never a single fake number.

Trigger milestone → Action re-run cone → Reward future self advances → Investment commitment to plan

The daily loop

Daily habit engine

HAB-04

One anchored ritual per day, built on the Hook model (Trigger → Action → Variable Reward → Investment). The whole loop completes in under two minutes and requires zero trades to “win.”

Trigger

06:45 briefing push

External: pre-open notification
→ Internal: “what did I miss overnight?”
Action

Read the 90-sec briefing

Regime → your deltas → one CLANK → one scar
Variable reward

Different every day

Some days calm, some days a rare CLANK cascade or a resolved prediction. Unpredictable value = the return pull.
Investment

Log one decision

Streak++ → calibration updates → tomorrow’s briefing gets smarter about you
The moat

The investment step is what compounds: every logged decision improves the personalization and the user’s own calibration record. After 30 days the user has a track record they can’t get anywhere else — the switching cost is their own history. That’s Bloomberg’s lock-in, earned honestly.

Premium upgrade psychology

PRM-08

The free tier must deliver a complete daily habit — the funnel is depth and history, not withholding the core loop. Users upgrade to see more of their own edge, not to unlock basic function.

Extending the terminal

Visual design language

VDL-09

Palantir restraint, BlackRock institutional clarity, Apple motion. Two accents only — green and red — over a near-black / true-white ground, so gamification reads as instrumentation, never arcade.

Palette white · black · green · red
INK · structure
GREEN · signal, competence
RED · risk, loss
QUIET · neutral data

No third hue. Green always means competence/positive; red always means risk. A neutral grey carries structural data so color stays meaningful.

Type system
DISPLAYSharper daily
BODYSF Pro / system, 1.62 line-height
DATA/LABELSCR-01 · CALIB 0.71

Light display weights + generous air (Apple). Monospace carries every ID, metric, eyebrow (Palantir). Tabular-nums wherever digits align.

Motion

Orchestrated, not scattered. Hero staggers up on load; sections reveal on scroll; data draws itself (the calibration sparkline). A streak tick pulses once; cards lift 4px on hover with a green edge. Everything honors prefers-reduced-motion. No confetti — premium comes from precision, not sparkle.

Component voice

Reuses the product’s glass PanelWrapper: mono header with an ID code, hairline border, faint glass gradient. Gamified widgets (belt, streak, health) are panels, visually equal to the quant panels — continuity, not a bolted-on game.
